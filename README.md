# MetaheuristicAlgorithmsPython

Various metaheuristic algorithms implemented in Python.

## Installation

Use ``pip3``:

```
pip3 install metaheuristic_algorithms_python
```

## Supported Platforms

* Python 3.4. 

It's not tested on Python 2.6 or 2.7 yet.
